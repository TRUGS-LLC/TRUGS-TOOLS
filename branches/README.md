# Experimental Branch Vocabularies

> **Under development.** CORE (the 7 node fields + 3 edge fields + hierarchy rules in TRUGS_PROTOCOL) is strong enough to cover most cases. These branches extend CORE with domain-specific vocabularies but are not yet fully mature.

Graduate to a top-level concept folder when mature enough to justify promotion (like AAA and EPIC already did).

## Branches here

| Branch | Status | Description |
|---|---|---|
| `law/` | DRAFT | Legal document structure (contracts, patents, policies). Originates in Xepayac/TRUGS-LAW. |
| `writer/` | DRAFT | Long-form writing (books, chapters, sections). |
| `knowledge/` | DRAFT | Knowledge graphs (concepts, entities, relations). Merged Living + Knowledge + Research + Decision vocabulary. |
| `nested/` | DRAFT | Recursive graph composition (tasks containing subgraphs). |
| `orchestration/` | DRAFT | Agent orchestration (AGENT, PRINCIPAL, PERMISSION, ESCALATION). |
| `mathgame/` | DRAFT | Adaptive math puzzle domain. |
